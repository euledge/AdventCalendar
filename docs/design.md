# デザインガイドライン

## 概要
本プロジェクトのデザインは、コミュニティアドベントカレンダーとしての「温かみ」「親しみやすさ」「クリスマスらしさ」を重視している。
提供されたロゴ画像 (`adventcalendar-logo.PNG`) をベースに、配色とトーン＆マナーを統一している。

## カラーパレット

| 用途 | カラーコード | 説明 |
|---|---|---|
| **背景色 (Background)** | `#f6eedb` | サイト全体の背景色。ロゴの背景色と一致させたクリーム色。 |
| **プライマリ (Primary)** | `#BA3627` | メインカラー。カレンダーのボード、ヘッダーのタイトル、強調テキストに使用される深い赤色。 |
| **セカンダリ (Secondary)** | `#FDF6E3` | サブカラー。装飾パーツのハイライトなどに使用。 |
| **テキスト (Text)** | `#BA3627` | タイトルや主要なテキストに使用。 |
| **カード背景 (Card Bg)** | `hsl(39, 100%, 93%)` | 投稿済みカードの背景色。サイト背景色と調和する明るいクリーム色。 |
| **ホバー (Hover)** | `hsl(148, 49%, 29%)` | 未登録カードのホバー時の背景色（クリスマスグリーン）。 |
| **無効 (Disabled)** | `#9CA3AF` | 未来の日付や無効な要素（Tailwind `gray-400` 相当）。 |

## タイポグラフィ
- **フォント**: ブラウザのデフォルトサンセリフフォントを使用（可読性重視）。
- **タイトル**: 太字 (`font-bold`) で強調し、トラッキング（文字間隔）をやや広め (`tracking-wider`) に設定。

## レイアウト構造
1.  **ヘッダー**:
    *   ロゴ画像を中央に配置。
    *   その下にイベントタイトルを配置。
    *   右上にユーザー情報とログイン/ログアウトボタンを配置。
    *   マージンを調整し、カレンダーとの一体感を高めている。
2.  **メインコンテンツ**:
    *   **カレンダーボード**: 赤色の角丸コンテナ (`rounded-3xl`)。
    *   **吊り下げ装飾**: ボード上部に円形の装飾を配置。
    *   **グリッドレイアウト**:
        *   **デスクトップ**: 7列（1週間）。曜日ヘッダーあり。月曜始まり（2025年12月1日に合わせる）。
        *   **タブレット**: 5列。
        *   **モバイル**: 2〜3列。
3.  **日付カード**:
    *   **アスペクト比**:
        *   デスクトップ: 正方形 (`aspect-[1/1]`)。
        *   その他: 横長 (`aspect-[3/2]`)。
    *   **クリスマスアイコン**: 右上に配置。CSSスプライトで25種類を表示。
    *   **日付バッジ**: 左上に配置。
    *   **情報表示**: 下部にユーザーアイコンと名前、記事タイトルを表示。

## UIコンポーネント
- **ツールチップ**:
    *   投稿済みカードにホバーすると、カード上部にポップアップ表示。
    *   記事タイトル全文、コメント、ユーザー情報を表示。
    *   カードの外側に表示されるオーバーレイタイプ。
- **ボタン**: `shadcn/ui` のボタンをベースにカスタマイズ。
- **ダイアログ**: 標準的なモーダルダイアログを使用。

## アセット
- `/adventcalendar-logo.PNG`: ヘッダーロゴ。
- `/tree-mikan.png`: ヘッダー左側のイラスト。
- `/gift-box.png`: ヘッダー右側のイラスト。
- `/icons/christmas-icons.jpg`: カレンダー日付用のアイコンスプライト画像。
