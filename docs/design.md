# デザインガイドライン

## 概要
本プロジェクトのデザインは、コミュニティアドベントカレンダーとしての「温かみ」「親しみやすさ」「クリスマスらしさ」を重視している。
提供されたロゴ画像 (`adventcalendar-logo.PNG`) をベースに、配色とトーン＆マナーを統一している。

## カラーパレット

| 用途 | カラーコード | 説明 |
|---|---|---|
| **背景色 (Background)** | `#f6eedb` | サイト全体の背景色。ロゴの背景色と一致させたクリーム色。 |
| **プライマリ (Primary)** | `#BA3627` | メインカラー。カレンダーのボード、ヘッダーのタイトル、強調テキストに使用される深い赤色。 |
| **セカンダリ (Secondary)** | `#FDF6E3` | サブカラー。装飾パーツのハイライトなどに使用。 |
| **テキスト (Text)** | `#BA3627` | タイトルや主要なテキストに使用。 |
| **カード背景 (Card Bg)** | `#FFFFFF` | カレンダーの日付カードの背景色。 |
| **無効 (Disabled)** | `#9CA3AF` | 未来の日付や無効な要素（Tailwind `gray-400` 相当）。 |

## タイポグラフィ
- **フォント**: ブラウザのデフォルトサンセリフフォントを使用（可読性重視）。
- **タイトル**: 太字 (`font-bold`) で強調し、トラッキング（文字間隔）をやや広め (`tracking-wider`) に設定。

## レイアウト構造
1.  **ヘッダー**:
    *   ロゴ画像を中央に配置。
    *   その下にイベントタイトルを配置。
    *   右上にユーザー情報とログイン/ログアウトボタンを配置（絶対配置またはFlexbox）。
2.  **メインコンテンツ**:
    *   **カレンダーボード**: 赤色の角丸コンテナ (`rounded-3xl`)。
    *   **吊り下げ装飾**: ボード上部に円形の装飾を配置し、壁掛けカレンダーのような見た目を演出。
    *   **グリッド**: 5列構成のカレンダーグリッド。
3.  **日付カード**:
    *   正方形 (`aspect-square`)。
    *   白背景で清潔感を出す。
    *   数字を中央に大きく配置。
    *   登録済みの場合は、ユーザーアイコンと記事タイトルを表示。

## UIコンポーネント
- **ボタン**: `shadcn/ui` のボタンをベースに、テーマカラーに合わせてカスタマイズ。
- **ダイアログ**: 標準的なモーダルダイアログを使用。テキストは全て日本語。
- **トースト通知**: `sonner` を使用し、成功時は緑、エラー時は赤で表示（デフォルト）。

## アセット
- `/adventcalendar-logo.PNG`: ヘッダーロゴとして使用。
- `/tree-mikan.png`: ヘッダー左側のクリスマスツリーとみかんのイラスト。
- `/gift-box.png`: ヘッダー右側のプレゼントボックスイラスト。

## ヘッダーのデザイン
- ロゴとタイトルを横並びにし、左側にツリー＆みかん、右側にプレゼントボックスを配置。
- タイトルフォントサイズを `text-5xl`（モバイル）/`text-7xl`（デスクトップ）に拡大。
- ホバー時のインタラクティブ効果を維持しつつ、全体の配色は赤・緑・オレンジでクリスマス感を演出。
